Bienvenue dans la Bonne Pioche

Afficher les règles du jeu

Objectif du jeu : Le but est de s'amuser avec un jeu à boire simple et sympa.

Déroulement :

  • Les joueurs entrent leur nom un par un.
  • Chaque joueur, tour à tour, devra deviner la valeur de la carte tirée (entre 1 et 10).
  • Une fois la valeur entrée, la carte est révélée.
    • Valeur incorrecte : Si le joueur n'a pas deviné la bonne valeur, le joueur suivant boit une gorgée.
    • Valeur correcte : Si le joueur devine correctement, personne ne boit.
  • Attention les têtes ont des actions différente
    • As (1) : Le joueur choisit une autre personne pour boire à sa place.
    • Valet (11) : Changement de sens de jeu.
    • Dame (12) : Le joueur suivant ne boit pas.
    • Roi (13) : Immunité de 2 tours pour le joueur suivant.
    • Joker (0) : Le joueur suivant doit finir son verre.
Fin du jeu : Le jeu se poursuit jusqu'à épuisement du jeu de cartes ou à la fin souhaitée par les joueurs.
